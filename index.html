Анимированная сортировка пузырьком

Описание использования

Для алгоритма сортировки пузырьком необходимо  выполнить следующие действия:

  1. Ввести значение элемента сортируемого поля массива в соотвествующее поле ниже
  2. Нажать кнопку "Добавить"
  3. При количестве элементов в массиве больше двух, нажать кнопку "Запустить сортировку"

После запуска алгоритма результат сортировки будет отображаться ниже в графе "Сортируемый массив". Каждый шаг итерации можно будет просмотреть в графе "Результаты сортировки"
